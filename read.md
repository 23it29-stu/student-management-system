# Student Management System

This project is the OOP version of a Student Management System.

## What it does

- Add, update, delete, and search student records
- Save data in MySQL
- Show performance and attendance reports
- Handle common database errors with user-friendly messages

## How to run

1. Open the `oop_version` folder.
2. Start MySQL and make sure the `student_management` database exists.
3. Check the credentials in `oop_version/database_manager.py`.
4. Run `python main.py`.

## Notes

- The root procedural version has been removed.
- The OOP version is the version intended for submission.

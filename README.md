# Student Management System

This repository contains the OOP version of a Student Management System built with Python, Tkinter, and MySQL.

## Main Features

- Add, update, delete, search, and view student records
- Store records in MySQL
- Generate student, performance, and attendance reports
- Show friendly error messages for database and record operations

## Run the Project

1. Open the `oop_version` folder.
2. Make sure MySQL is running and the `student_management` database exists.
3. Update the database credentials in `oop_version/database_manager.py` if needed.
4. Run `python main.py` from inside `oop_version`.

## Entry Point

The official submission is the OOP implementation in `oop_version/main.py`.
